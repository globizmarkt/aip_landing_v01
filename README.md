## Doctrinas Activas

| Doctrina | Descripción |
|----------|-------------|
| R0 — Agnosticismo Radical | El Core no conoce el negocio. Solo renderiza `offeringConfig`. |
| R2 — Light DOM | Prohibido `attachShadow()`. Todos los componentes en Light DOM. |
| R3 — Zero-Hex | Cero colores hexadecimales inline. Solo `var(--theme-*)`. |
| R4 — i18n Strict | Cero texto hardcodeado. Todo vía `data-i18n`. |
| R5 — Soberanía AIP | Nomenclatura y paleta institucional innegociable. |
